# GameGetter
Download a game or other file to your NDS/DSI with http:

Start a webserver on your host machine. Use a a game cart that uses wifi to save a WFC connection in the first slot. Re-insert your flashcat.  Make a simple index.html web page with a href link to your file and save it.
Copy gamegetter.nds to your DS/DSI flashcart sd card or DSI sd card. Run the program. The game or other file will download automatically to the root of your sd card. Power your device off and on again and play the game.
